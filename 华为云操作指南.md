# 华为云 ModelArts 操作指南（从零开始）

记录一下整个流程，方便复现。环境是华为云 ModelArts 的 Notebook（昇腾 NPU）。

## 1. 委托授权（第一次用才要）

ModelArts 第一次用会要求"添加授权"：

- 授权对象类型选「所有用户」
- 委托选择点「新增委托」
- 权限模板选「(全选)」，下面服务列表保持默认
- 点「创建」

授权一次即可，之后不再弹。

## 2. 创建 Notebook 实例

进入 ModelArts -> 开发环境 -> Notebook -> 创建：

- 自动停止：开启，1 小时（防止忘记关）
- 镜像：选 PyTorch，右上角切 **ARM**，版本 `pytorch_2.6.0-cann_8.2.rc1-py_3.11-...-aarch64-snt9b`
- 资源池：公共资源池
- 规格：NPU 标签下选 `1*ascend-snt9b1`（1 卡）
- 存储：云硬盘 EVS，10 GB

> 注意：NPU 规格只在部分区域有货。如果某区域「暂无资源」，用弹窗里的「切换区域」换一个再试。

点「创建」，等几分钟实例变成「运行中」，点「打开」进 JupyterLab。

## 3. 拉代码

JupyterLab 里开一个 Terminal，执行：

```bash
git clone https://github.com/Elwaysss/NeuralNetwork_Project2.git
cd NeuralNetwork_Project2
```

## 4. 确认 NPU 可用

```bash
python test_ascend_npu.py
```

看到最后一行「Ascend NPU 基本可用性测试完成」就 OK。

## 5. 跑实验

```bash
# 任务一（CIFAR-10 分类）
cd part1_cifar10 && bash run.sh && cd ..

# 任务二（BatchNorm）
cd codes/VGG_BatchNorm && bash run.sh && cd ..
```

数据集会自动下载。想省时间可以先把 epoch 调小：

```bash
BASE_EPOCHS=20 ABL_EPOCHS=15 bash part1_cifar10/run.sh
```

## 6. 取结果

- 图在各自的 `figures/` 目录，数值结果在 `results/`
- 在 JupyterLab 左侧右键文件 -> Download 下载下来
- 模型权重（`*.pth`）记得下载后传网盘，链接写进报告

## 7. 用完关实例

回 Notebook 列表，把实例「停止」，停了基本就不计费了。
